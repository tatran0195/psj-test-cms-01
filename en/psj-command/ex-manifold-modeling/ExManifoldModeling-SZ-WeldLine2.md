## Description

## Syntax

```psj
ExManifoldModeling.SZ.WeldLine2(crlFaces, crlParts, dLayerWidth, iLayerNumber)
```

Ribbon: <menuselection>ExManifoldModeling &#187; SZ &#187; WeldLine2</menuselection>

## Inputs

### `crlFaces`

- A _List of Cursor_ specifying the face.
- This is a required input.

### `crlParts`

- A _List of Cursor_ specifying the part.
- This is a required input.

### `dLayerWidth`

- A _Double_ specifying the layer width.
- This is a required input.

### `iLayerNumber`

- An _Integer_ specifying the layer number.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.
