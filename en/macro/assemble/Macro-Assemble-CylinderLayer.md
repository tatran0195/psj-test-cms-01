## Description

Cylinder Layer

## Syntax

```psj
CylinderLayer(cursor crFace, cursor crNode)
```

## Inputs

### `1. Cursor`

Face cursor(6:Face ID)

### `2. Cursor`

Node cursor(10:Node ID)

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
CylinderLayer(6:8, 10:478)
```
