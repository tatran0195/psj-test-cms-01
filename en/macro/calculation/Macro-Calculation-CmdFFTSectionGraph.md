## Description

Make the section graph.

## Syntax

```psj
CmdFFTSectionGraph(int iPointIndex)
```

## Inputs

### `1. int`

- An Integer specifying the point on the layer to make the section graph.

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
CmdFFTSectionGraph(1)
```
