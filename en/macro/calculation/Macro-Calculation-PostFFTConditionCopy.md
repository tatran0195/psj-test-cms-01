## Description

Copy a FFT analysis.

## Syntax

```psj
PostFFTConditionCopy(cursor crTarget)
```

## Inputs

### `1. cursor`

- A Cursor specifying the copy source.

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
PostFFTConditionCopy(220:1)
```
