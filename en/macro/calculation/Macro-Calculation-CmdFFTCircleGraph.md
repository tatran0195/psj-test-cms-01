## Description

Display the circle graph.

## Syntax

```psj
CmdFFTCircleGraph(int iLayer)
```

## Inputs

### `1. int`

- An Integer specifying the layer of mode to make the circle graph.

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
CmdFFTCircleGraph(1)
```
