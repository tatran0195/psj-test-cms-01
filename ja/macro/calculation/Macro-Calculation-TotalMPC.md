## Description

Display a graph (a chart) of total MPC.

## Syntax

```psj
TotalMPC(crResponse)
```

## Inputs

### `1. cursor`

- A Cursor specifying the response to calculate total MPC.

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
TotalMPC(0:0)
```
