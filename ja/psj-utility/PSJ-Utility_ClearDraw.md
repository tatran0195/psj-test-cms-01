## Description

Clear all the drawings in Main Window.

## Syntax

```psj
JPT.ClearDraw()
```

## Inputs

This utility function does not require any input value.

## Return Code

- A _Boolean_ specifying Succeeded or Failed.
    - True: Succeeded.
    - False: Failed.
