## Description

Load JTDB file

## Syntax

```psj
FileMenu.Open(strFileName="", bUseTmpTable=False)
```

Ribbon: <menuselection>FileMenu &#187; Open</menuselection>

## Inputs

### `strFileName`

- A _String_ specifying the file name.
- The default value is "".

### `bUseTmpTable`

- A _Boolean_ specifying the use temporary table.
- The default value is False.

## Return Code

A String of 1 if success, or 0 if fail.
