## Description

Save file JTDB

## Syntax

```psj
FileMenu.Save(strFileName="")
```

Ribbon: <menuselection>FileMenu &#187; Save</menuselection>

## Inputs

### `strFileName`

- A _String_ specifying the file name.
- The default value is "".

## Return Code

A String of 1 if success, or 0 if fail.
