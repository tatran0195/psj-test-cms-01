## Description

Suppress part on Assembly tree.

## Syntax

```psj
Assembly.RightClick.Suppress(...)
```

Ribbon: <menuselection>Assembly &#187; Right Click &#187; Suppress</menuselection>

## Inputs

### `crlParts`

- A _List of Cursor_ specifying the part.
- The default value is [].

## Return Code

_True_ if success, or _False_ if fail.
