## Description

Create LS-Dyna Analysis Job.

## Syntax

```psj
Analysis.LSDYNAJob(...)
```

Ribbon: <menuselection>Analysis &#187; LSDYNAJob</menuselection>

## Inputs

### `crEdit`

- A _Cursor_ specifying the existed analysis job to modify it instead of creating a new ones.
- The default value is None.

## Return Code

A String of 1 if success, or 0 if fail.
